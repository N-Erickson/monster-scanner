# Monster Scanner

**A real-time monster detection dashboard that continuously scans every room in your house and confirms: there are absolutely NO monsters. Not even one.**

Built for bedtime. Built for peace of mind. Built because sometimes kids need to see the proof.

## What It Does

Monster Scanner is a single-page web app that looks and feels like a legit high-tech scanning system. It runs continuously, cycling through every room in the house with a radar sweep, live status updates, and a scrolling activity log. Every room always comes back the same: **SAFE!**

### Features

- Spinning radar with a protective shield glow and room detection dots
- Room-by-room scanning: Bedroom, Closet, Under the Bed, Bathroom, Hallway, Kitchen, Living Room, Garage
- Live scrolling log with detailed check-by-check updates
- Safety Report dashboard: Monsters Found (always 0), Rooms Safe (always 8/8), Scanner Power, Protection Shield
- Rotating reassurance banner with comforting messages
- Fully responsive — fills any screen from a phone to a TV
- Night-sky color scheme with soft cyan and green accents (not scary at all)
- Runs forever with no interaction needed — just open it and leave it on

## How to Use

1. Open `index.html` in any browser
2. That's it. The scanner starts immediately.
3. Leave it running on a tablet, laptop, or old phone next to the bed.

No server needed. No install. No dependencies. Just one HTML file.

## The Tech

Pure HTML + CSS + JavaScript in a single file. Uses Google Fonts (Orbitron + Nunito) for that high-tech-but-friendly look. CSS animations handle the radar sweep, shield pulse, and all the glowing effects. The scanning logic is a simple async loop that will happily run all night.

## Why

Because monsters aren't real, but sometimes you need a cool-looking scanner to prove it.
